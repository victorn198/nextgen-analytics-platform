# Workflow Playbooks

Files in this directory are reference playbooks for the agent.

They are not automatically exposed as slash commands by `codex-acp`.

Use them as local instructions:

- open the relevant file
- follow its checklist or command sequence
- report outcomes in the current chat

Examples:

- planning: `plan.md`
- validation: `validate.md`
- review: `code-review.md`
- pipeline execution: `pipeline.md`
- debugging: `debug.md`
- performance review: `perf.md`
